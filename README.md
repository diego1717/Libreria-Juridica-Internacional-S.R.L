# Libreria-Juridica-Internacional-S.R.L

Sitio de libreria juridica con enfoque editorial, area de tienda y catalogo comercial.

## Compatibilidad con Hostinger

- La raiz del repo ahora esta preparada como proyecto `Vite`, que Hostinger puede detectar al importar.
- Build recomendado en Hostinger: `npm run build`
- Carpeta de salida: `dist`

## Ver la pagina

- Desarrollo local: `npm install` y luego `npm run dev`
- Build local: `npm run build`
- Si solo quieres revisar rapido la maqueta, tambien puedes abrir `index.html` en el navegador.

## Estructura

- `index.html`: contenido y secciones principales.
- `styles.css`: sistema visual rojo/editorial y responsive layout.
- `main.js`: render del catalogo, filtros, busqueda, orden y panel de pedido.
- `package.json`: manifiesto para Vite y Hostinger.
- `vite.config.js`: configuracion con salida estatica portable.

## Referencias de diseno

- `impeccable-main/`: referencia visual/editorial.
- `ui-ux-pro-max-skill-main/`: referencia de sistema de diseno y criterio UX.
