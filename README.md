# Libreria-Juridica-Internacional-S.R.L

Landing page estatica para una libreria juridica con enfoque editorial y venta de libros.

## Ver la pagina

- Abre `index.html` en el navegador.

## Estructura

- `index.html`: contenido y secciones principales.
- `styles.css`: sistema visual y responsive layout.
- `main.js`: filtros del catalogo y panel de pedido rapido.

## Referencias de diseno

- `impeccable-main/`: referencia visual/editorial.
- `ui-ux-pro-max-skill-main/`: referencia de sistema de diseno y criterio UX.
